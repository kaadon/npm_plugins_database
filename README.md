<p align="center">database client for the node.js</p>

### MYSQL配置
> ###### json 配置
> ```json
>     {
>         "MYSQL":{
>             "HOST":"127.0.0.1",
>             "PORT":"3306",
>             "NAME":"mysql",
>             "USER":"mysql",
>             "PWD":"mysql"
>         }
>     }
> ```
> 
> ###### dotenv 配置
> ```dotenv
> #dotenv
> MYSQL='{"HOST":"127.0.0.1","PORT":"3306","NAME":"mysql","USER":"mysql","PWD":"mysql"}'
> ```
>
